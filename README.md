# fufu_sandbox
Vagrant Sandbox for Development Fufumart ID Internal Tools

### In this box
- Ubuntu 22.04 LTS
- Ansible
- Git
- Nginx
- PHP 7.4
- MySQL 8.0
- NVM
- NodeJS 18.17.0

### Getting Started
- Up and Running
  - vagrant up
- Stop Vagrant
  - vagrant halt
- Reload Vagrant
  - vagrant reload
 
- Login to SSH Console
  - vagrant ssh
  - ssh vagrant@192.168.33.10
 
### Enjoy
